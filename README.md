**Movie Library** — это десктопное приложение на Python для ведения личной коллекции фильмов. Программа позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, фильтровать их по различным критериям, а также сохранять все данные в JSON-файл.

### Ключевые возможности:
- ✚ Добавление фильмов с валидацией данных
- 🔍 Фильтрация по жанру и году выпуска
- 💾 Автоматическое сохранение в JSON
- 📊 Статистика коллекции (средний рейтинг, лучший/худший фильм)
- 🎨 Интуитивно понятный интерфейс на tkinter
- ✅ Встроенные модульные тесты
